# Scientist traits

This page lists traits specific to the Scientist class and its sub-classes only. For common traits, please see leader traits . Traits represent the background and characteristics of a leader. Most traits have multiple tiers and can be upgraded. The way in which leaders gain or upgrade traits depends on whether the Galactic Paragons DLC is enabled:

| Skill level | With Galactic Paragons | Without Galactic Paragons |
|---|---|---|
| | Basic trait | |
| | Basic trait | Basic or veteran trait |
| | Veteran class If gestalt ruler or node, veteran councilor trait instead | |
| | Veteran trait | Basic or veteran trait |
| | Veteran trait | |
| | Veteran trait | Basic or veteran trait |
| | Destiny trait If gestalt or paragon with destiny trait, veteran trait instead | |
| | Veteran trait | Basic or veteran trait |
| | Veteran trait | |
| Mechanics | - Traits can be selected manually<br>- Veteran traits cap at Veteran trait III, tier 3<br>- Can set traits to be selected automatically | - Traits are selected automatically<br>- Veteran traits cap at tier 2<br>- Most veteran traits are not available<br>- A few additional basic traits are available |

## Veteran traits

With the Galactic Paragons DLC, leaders that reach skill level 4 can select a veteran class, which will unlock a pool of veteran traits selectable on the next level-ups. Without the Galactic Paragons DLC, leaders do not use veteran classes, but can randomly gain some of these traits at level-up if their requirements are met.

Each scientist can only gain one expertise trait from each research area when leveling up, but more expertise traits from the same area can be added by events. If the empire has the Technocracy civic, each scientist starts with a random tier I expertise trait, and the traits gain additional effects when governing a planet or sector.

Without the Galactic Paragons DLC, they can be gained randomly upon level-up if their requirements are met.
